# Prompt Quizzer – JavaScript Quiz Game

## Description
Prompt Quizzer is a simple JavaScript-based quiz game that runs entirely in the browser console. It uses `prompt()` for input and `alert()` for feedback. The game asks predefined questions, checks answers, tracks the score, and displays the final result.

## Features
- Runs completely in the browser console
- Uses arrays, loops, functions, and conditionals
- Case-insensitive answer checking using `toLowerCase()` and `trim()`
- Immediate feedback after each question
- Final score display

## How to Run
1. Open any web browser
2. Open Developer Tools → Console
3. Copy and paste the code from 'lab4.js'
4. Press Enter to start the quiz
